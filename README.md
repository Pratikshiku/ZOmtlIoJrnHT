### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（访客）、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 访客预约管理
6. 访客来访管理


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 访客管理
4. 访客预约管理
5. 访客来访管理
